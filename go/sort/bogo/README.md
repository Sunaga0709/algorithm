# ボゴソート

## 概要
ボゴソートは、配列をランダムに並べ替え続けて、たまたま整列するまで待つソートです。実用には向きませんが、確率的な発想を知る教材になります。`sort/bogo` はその性質を学ぶ場所です。

## 使いどころ
- ソートの冗談みたいな例を見たいとき
- ランダム性と探索の関係を知りたいとき
- 実用では使わないアルゴリズムの例として学びたいとき

## 手順
1. 配列をランダムにシャッフルする。
2. 昇順になっているか調べる。
3. 整列済みなら終わる。
4. そうでなければまたシャッフルする。
5. 失敗回数に上限を設けることもある。

## 変化の例
`[3, 1, 2]` を考える。

```text
1回目: [1, 3, 2]
2回目: [2, 1, 3]
3回目: [1, 2, 3]
```

## 実装ポイント
- 整列判定は毎回必要。
- いつ終わるか予測できない。
- 上限を置かないと終わらない場合がある。
- 学習用以外では使わない。

## 計算量
| 操作 | 計算量 |
| --- | --- |
| 平均 | 非常に大きい |
| 最悪 | 無限に近い |
| 追加メモリ | O(1) |
