# バケットソート

## 概要
バケットソートは、値の範囲をいくつかの入れ物に分け、各入れ物を個別に整列してから結合する方法です。値の分布がわかっているときに強いです。`sort/bucket` はその基本を学ぶ場所です。

## 使いどころ
- 値の範囲がだいたい決まっているとき
- 分布が比較的均一なとき
- 桶に分けてから別の小さなソートを使いたいとき

## 手順
1. 値の範囲を決める。
2. 範囲ごとにバケットを作る。
3. 各値を対応するバケットへ入れる。
4. 各バケットの中を整列する。
5. バケットを順に連結する。

## 変化の例
`0 から 99` の整数を 10 個ずつ分ける。

```text
0-9   : [3, 7]
10-19 : [12]
20-29 : [25, 28]
30-39 : []
```

各バケットを整列してから

```text
[3, 7, 12, 25, 28]
```

## 実装ポイント
- バケット数の決め方が重要。
- 各バケット内のソート方法も選ぶ必要がある。
- 分布が偏ると速くならない。
- 実装では範囲外の値をどう扱うかを決める。

## 計算量
| 条件 | 計算量 |
| --- | --- |
| 平均 | O(n + k) 近辺 |
| 最悪 | バケットの偏り次第で悪化 |
| 追加メモリ | O(n + k) |
