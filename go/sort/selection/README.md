# 選択ソート

## 概要

選択ソートは、未整列部分から最小値を探して先頭に置く処理を繰り返すソートです。前から順に確定していきます。`sort/selection` はその基本を学ぶ場所です。

## 使いどころ

- 実装を単純に保ちたいとき
- 交換回数を少なめにしたいとき
- 小さい配列を扱うとき

## 手順

1. 未整列部分の先頭を `i` とする。
2. `i` 以降から最小値の位置を探す。
3. その最小値を `i` と交換する。
4. `i` を 1 つ進める。

## 変化の例

`[5, 2, 4, 1]` を昇順にする。

```text
初期: [5, 2, 4, 1]
1回目: [1, 2, 4, 5]
2回目: [1, 2, 4, 5]
3回目: [1, 2, 4, 5]
```

## 実装ポイント

- 各周回で最小値の位置を必ず更新する。
- 交換は 1 回の周回につき 1 回だけ。
- 安定ソートではない。
- データがほぼ整列済みでも比較回数は減りにくい。

## 計算量

| 操作       | 計算量 |
| ---------- | ------ |
| 平均       | O(n^2) |
| 最悪       | O(n^2) |
| 追加メモリ | O(1)   |
