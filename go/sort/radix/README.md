# 基数ソート

## 概要
基数ソートは、1 の位、10 の位、100 の位のように桁ごとに並べ替えるソートです。各桁では安定ソートを使うのが基本です。`sort/radix` はその考え方を学ぶ場所です。

## 使いどころ
- 非負整数をまとめて整列したいとき
- 桁数がある程度そろっているとき
- 比較ソート以外の方法を学びたいとき

## 手順
1. 1 の位から始める。
2. その桁の値で安定に並べ替える。
3. 次の桁へ進む。
4. 最大桁まで繰り返す。
5. 最後に全体が整列する。

## 変化の例
`[170, 45, 75, 90, 802, 24, 2, 66]` を考える。

```text
1の位後:  [170, 90, 802, 2, 24, 45, 75, 66]
10の位後: [802, 2, 24, 45, 66, 170, 75, 90]
100の位後:[2, 24, 45, 66, 75, 90, 170, 802]
```

## 実装ポイント
- 桁ごとの処理は安定でなければならない。
- 桁の最大数を先に調べる。
- 10 進以外の基数でも書ける。
- 負の値を扱うなら別の工夫がいる。

## 計算量
| 条件 | 計算量 |
| --- | --- |
| 平均 | O(d(n + k)) |
| 最悪 | O(d(n + k)) |
| 追加メモリ | 実装次第 |
