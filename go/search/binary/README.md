# 二分探索

## 概要
二分探索は、ソート済みの配列を中央から二分して目的値を探す方法です。範囲を半分ずつ減らせるので速いです。`search/binary` はその基本を学ぶ場所です。

## 使いどころ
- 配列が昇順または降順に整列しているとき
- 何度も同じ集合から探したいとき
- 線形探索より速くしたいとき

## 手順
1. 左端 `left` と右端 `right` を決める。
2. 真ん中 `mid` を求める。
3. `mid` の値と探したい値を比べる。
4. 小さければ右半分を、 大きければ左半分を捨てる。
5. `left > right` になったら見つからない。

## 変化の例
配列:
```text
[2, 4, 7, 10, 14, 18, 21]
```

`14` を探す。

```text
left=0, right=6, mid=3 -> 10
``` 

`14` は大きいので右へ。

```text
left=4, right=6, mid=5 -> 18
```

`14` は小さいので左へ。

```text
left=4, right=4, mid=4 -> 14
```

## 実装ポイント
- 配列は必ず整列済みでなければならない。
- `mid` は `left + (right-left)/2` の形にすると安全。
- 再帰でも反復でも書ける。
- 見つけた位置を返すか、真偽値を返すかを先に決める。

## 計算量
| 操作 | 計算量 |
| --- | --- |
| 探索 | O(log n) |
| 追加整列済み配列への挿入 | O(n) |
