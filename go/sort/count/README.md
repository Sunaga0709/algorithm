# 計数ソート

## 概要

計数ソートは、値の出現回数を数えて、その情報から整列結果を作るソートです。比較をほとんど使いません。`sort/count` はその基本を学ぶ場所です。

## 使いどころ

- 整数の範囲が狭いとき
- 比較回数を減らしたいとき
- 値の頻度を数える処理と相性がよいとき

## 手順

1. 値の最大値を知る。
2. 出現回数の配列を作る。
3. 各値の回数を数える。
4. 累積和を取ると、位置がわかる。
5. その位置に値を置く。

## 変化の例

`[4, 2, 2, 8, 3]` を考える。

```text
count
2: 2
3: 1
4: 1
8: 1

result
[2, 2, 3, 4, 8]
```

## 実装ポイント

- 値の範囲が広すぎると配列が大きくなる。
- 負の値があるなら、ずらして扱う。
- 安定性を保つなら後ろから詰める方法がよい。
- 小さい整数範囲で力を発揮する。

## 計算量

| 条件       | 計算量   |
| ---------- | -------- |
| 平均       | O(n + k) |
| 最悪       | O(n + k) |
| 追加メモリ | O(n + k) |
