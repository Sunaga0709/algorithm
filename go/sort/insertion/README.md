# 挿入ソート

## 概要

挿入ソートは、左側の整列済み部分に新しい要素を 1 つずつ差し込むソートです。手札を並べるように考えるとわかりやすいです。`sort/insertion` はその基本を学ぶ場所です。

## 使いどころ

- ほぼ整列済みの配列を扱うとき
- 小さい配列を扱うとき
- 安定ソートが欲しいとき

## 手順

1. 左端の 1 個を整列済みとみなす。
2. 次の要素を取り出す。
3. 左側の大きい要素を 1 つ右へずらす。
4. 空いた場所に取り出した要素を入れる。
5. これを右端まで繰り返す。

## 変化の例

`[5, 3, 4, 1]` を昇順にする。

```text
初期: [5, 3, 4, 1]
1回目: [3, 5, 4, 1]
2回目: [3, 4, 5, 1]
3回目: [1, 3, 4, 5]
```

## 実装ポイント

- 右から左へ比較すると差し込みやすい。
- 途中で止まれるので、ほぼ整列済みなら速い。
- 同じ値の順序を保ちやすい。
- ずらし処理と挿入位置の更新を分けて考えるとよい。

## 計算量

| 操作       | 平均   | 最悪   | ほぼ整列済み |
| ---------- | ------ | ------ | ------------ |
| 挿入ソート | O(n^2) | O(n^2) | O(n) に近い  |
