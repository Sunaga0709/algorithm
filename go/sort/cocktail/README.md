# カクテルソート

## 概要
カクテルソートは、バブルソートを左右両方向に行うソートです。前からも後ろからも泡が動くので、両端の要素をまとめて整えやすいです。`sort/cocktail` はその基本を学ぶ場所です。

## 使いどころ
- バブルソートの改良版を見たいとき
- 小さい配列を教材として扱うとき
- 両方向の走査を理解したいとき

## 手順
1. 左から右へ隣同士を比べて交換する。
2. 右端に最大値を送る。
3. 今度は右から左へ比べて交換する。
4. 左端に最小値を送る。
5. 範囲を内側へ狭めながら繰り返す。

## 変化の例
`[4, 2, 3, 1]` を整列する。

```text
forward:  [2, 3, 1, 4]
backward: [1, 2, 3, 4]
```

## 実装ポイント
- 前後の走査で未確定範囲を縮める。
- 交換がなければ早期終了できる。
- バブルソートより少しだけ効率がよいことがある。
- 実装時は走査範囲の更新を間違えやすい。

## 計算量
| 操作 | 計算量 |
| --- | --- |
| 平均 | O(n^2) |
| 最悪 | O(n^2) |
| 追加メモリ | O(1) |
