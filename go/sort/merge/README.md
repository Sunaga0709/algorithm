# マージソート

## 概要

マージソートは、配列を半分に分け、分割した配列を整列しながら結合するソートです。分割統治の代表例です。`sort/merge` はその基本を学ぶ場所です。

## 使いどころ

- 安定ソートが欲しいとき
- 大きい配列を安定して整列したいとき
- 再帰と分割統治を学びたいとき

## 手順

1. 配列を中央で 2 つに分ける。
2. それぞれを再帰的に整列する。
3. 2 つの整列済み配列を先頭から比べる。
4. 小さいほうを結果配列に入れる。
5. 片方が尽きたら残りをそのまま入れる。

## 変化の例

`[5, 2, 4, 1]` を整列する。

```text
split
[5, 2]   [4, 1]

sort
[2, 5]   [1, 4]

merge
[1, 2, 4, 5]
```

## 実装ポイント

- 再帰の終了条件は要素数 1。
- マージ時は 2 本のポインタを使う。
- 安定性を保つには、同値なら左側を先に取る。
- 追加配列が必要になることが多い。

## 計算量

| 操作       | 計算量     |
| ---------- | ---------- |
| 平均       | O(n log n) |
| 最悪       | O(n log n) |
| 追加メモリ | O(n)       |
