# スタック

## 概要

スタックは後入れ先出し、つまり最後に入れた要素を最初に出す構造です。皿を積むようなイメージです。`stack-quere/stack` は基本動作を学ぶ場所です。

## 使いどころ

- 直前の状態を戻したいとき
- 式の評価や再帰の代わりに使いたいとき
- DFS のように深く進む探索で使いたいとき

## 手順

1. `push` で末尾に積む。
2. `pop` で末尾から取り出す。
3. `top` で最後の要素だけ見る。
4. 空のときは `pop` しない。

## 変化の例

初期状態:

```text
bottom [1] [2] [3] top
```

`push(4)`:

```text
bottom [1] [2] [3] [4] top
```

`pop()`:

```text
bottom [1] [2] [3] top
```

## 実装ポイント

- 配列で作るときは末尾をスタックの頂点にする。
- `push` と `pop` は同じ側で行う。
- 空判定は長さ 0 で見る。
- 再帰の呼び出し順と相性がよい。

## 計算量

| 操作 | 計算量 |
| ---- | ------ |
| push | O(1)   |
| pop  | O(1)   |
| top  | O(1)   |
| 検索 | O(n)   |
