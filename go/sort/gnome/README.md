# グノームソート

## 概要
グノームソートは、隣同士を見て順序が逆なら 1 つ戻り、正しければ 1 つ進むソートです。庭の小人が鉢を並べ直す話になぞらえられます。`sort/gnome` はその基本を学ぶ場所です。

## 使いどころ
- 挿入ソートに近い流れを別の形で見たいとき
- 実装を単純に保ちたいとき
- 学習用に小さな例を追いたいとき

## 手順
1. 先頭から始める。
2. となり合う 2 要素を比べる。
3. 順序が正しければ 1 つ進む。
4. 順序が逆なら交換して 1 つ戻る。
5. 先頭まで来たら、そのまま進む。

## 変化の例
`[4, 3, 2, 1]` を整列する。

```text
[4, 3, 2, 1]
[3, 4, 2, 1]
[3, 2, 4, 1]
[2, 3, 4, 1]
[2, 3, 1, 4]
[2, 1, 3, 4]
[1, 2, 3, 4]
```

## 実装ポイント
- 1 つ戻るのが特徴なので、位置管理が大事。
- 挿入ソートより説明しやすい場合がある。
- 安定ソートとして扱える。
- かなり遅いので実用性は高くない。

## 計算量
| 操作 | 計算量 |
| --- | --- |
| 平均 | O(n^2) |
| 最悪 | O(n^2) |
| 追加メモリ | O(1) |
