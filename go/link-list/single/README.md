# 単方向連結リスト

## 概要
単方向連結リストは、各ノードが次のノードだけを知っているリストです。前には戻れませんが、構造がシンプルで実装しやすいです。`link-list/single` はその基本を学ぶための場所です。

## 使いどころ
- 要素の順番を保って並べたいとき
- 先頭から順に走査できれば十分なとき
- 実装をできるだけ軽くしたいとき

## 手順
1. ノードに `value` と `next` を持たせる。
2. リストは先頭の `head` を持つ。
3. 先頭へ追加するときは、新ノードの `next` を旧 `head` にする。
4. そのあと `head` を新ノードへ更新する。
5. 途中へ入れるときは、挿入位置の直前まで進む。
6. 直前ノードの `next` を新ノードにし、新ノードの `next` を元の後続にする。

## 変化の例
初期状態:
```text
head -> [10] -> [20] -> [30] -> nil
```

先頭に `5` を入れる:
```text
head -> [5] -> [10] -> [20] -> [30] -> nil
```

`20` のあとに `25` を入れる:
```text
head -> [5] -> [10] -> [20] -> [25] -> [30] -> nil
```

`10` を消す:
```text
head -> [5] -> [20] -> [25] -> [30] -> nil
```

## 実装ポイント
- 先頭を消すときは `head = head.next` で済む。
- 途中削除では、必ず一つ前のノードを覚えておく。
- 末尾操作は、毎回先頭からたどると O(n) になる。
- `nil` に当たったら走査終了と考える。

## 計算量
| 操作 | 計算量 |
| --- | --- |
| 先頭挿入 | O(1) |
| 先頭削除 | O(1) |
| 途中挿入 | O(n) |
| 途中削除 | O(n) |
| 検索 | O(n) |
