# シェルソート

## 概要

シェルソートは、離れた要素どうしを先に整えてから、間隔を小さくしながら仕上げるソートです。挿入ソートの弱点を減らしやすいのが特徴です。`sort/shell` はその基本を学ぶ場所です。

## 使いどころ

- 挿入ソートより速くしたいとき
- そこそこ小さい配列を扱うとき
- 間隔を使った整列の考え方を学びたいとき

## 手順

1. 大きな `gap` を決める。
2. `gap` 離れた要素どうしを比較する。
3. 順序が逆なら入れ替える。
4. `gap` を小さくして再び同じ処理をする。
5. 最後は `gap = 1` で挿入ソートの形にする。

## 変化の例

`[8, 5, 3, 7, 6, 2, 1, 4]` を考える。

```text
gap=4: [6, 2, 1, 4, 8, 5, 3, 7]
gap=2: [1, 2, 3, 4, 6, 5, 8, 7]
gap=1: [1, 2, 3, 4, 5, 6, 7, 8]
```

## 実装ポイント

- `gap` の並びは実装ごとに少し違う。
- 最後の `gap=1` が仕上げになる。
- 比較回数を減らしやすいが、安定ソートではない。
- 挿入ソートの拡張版として考えると理解しやすい。

## 計算量

| 条件       | 計算量       |
| ---------- | ------------ |
| 平均       | 間隔列による |
| 最悪       | O(n^2)       |
| 追加メモリ | O(1)         |
