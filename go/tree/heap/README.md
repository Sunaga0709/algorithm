# ヒープ木

## 概要

ヒープは、親と子の大小関係を保つ完全二分木です。最大値または最小値をすぐ取り出せます。`tree/heap` はその基本を学ぶ場所です。

## 使いどころ

- 最大値や最小値を何度も取り出したいとき
- 優先度つきの処理をしたいとき
- ヒープソートの土台にしたいとき

## 手順

1. 末尾に新しい要素を置く。
2. 親と比べて、必要なら上へ持ち上げる。
3. 根を取り出すときは末尾の要素を根に移す。
4. 子と比べて、必要なら下へ落とす。
5. これで最大ヒープまたは最小ヒープを保つ。

## 変化の例

最大ヒープに `10, 20, 5, 30` を入れる。

```text
        [10]
       /
    [20]
```

`20` を入れたあと上へ持ち上げる。

```text
        [20]
       /
    [10]
```

`30` を入れるとさらに根へ来る。

```text
        [30]
       /   \
    [10]   [20]
```

## 実装ポイント

- 配列で表すと、親子の位置計算が簡単。
- 0 始まりなら親は `(i-1)/2`、左子は `2*i+1`、右子は `2*i+2`。
- `push` は上へ、`pop` は下へ調整する。
- 完全二分木の形を崩さない。

## 計算量

| 操作       | 計算量   |
| ---------- | -------- |
| 挿入       | O(log n) |
| 最大値取得 | O(1)     |
| 取り出し   | O(log n) |
