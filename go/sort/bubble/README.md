# バブルソート

## 概要
バブルソートは、隣り合う要素を比べて順序が逆なら入れ替える処理を何度も繰り返すソートです。大きい値が右へ少しずつ流れていくのが特徴です。`sort/bubble` はその基本を学ぶ場所です。

## 使いどころ
- 小さい配列を扱うとき
- 実装をとても単純にしたいとき
- 教材として交換の流れを見たいとき

## 手順
1. 左から右へ隣同士を比較する。
2. 左のほうが大きければ入れ替える。
3. 1 回の走査で最大値が末尾に落ち着く。
4. 末尾を 1 つずつ確定させながら繰り返す。

## 変化の例
`[5, 3, 4, 1]` を昇順にする。

```text
初期: [5, 3, 4, 1]
1回目: [3, 5, 4, 1]
2回目: [3, 4, 5, 1]
3回目: [3, 4, 1, 5]

次の周回:
        [3, 4, 1, 5]
        [3, 1, 4, 5]
        [1, 3, 4, 5]
```

## 実装ポイント
- 内側のループは `0` から未確定部分の終端まで回す。
- 交換が一度も起きなければ早く終われる。
- 安定ソートなので、同じ値の順序は保ちやすい。

## 計算量
| 操作 | 計算量 |
| --- | --- |
| 平均 | O(n^2) |
| 最悪 | O(n^2) |
| 追加メモリ | O(1) |
